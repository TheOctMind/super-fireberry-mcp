# Contributing to Super Fireberry MCP

First off, thank you for considering contributing to Super Fireberry MCP! It's people like you that make this tool great.

## Code of Conduct

By participating in this project, you are expected to uphold our Code of Conduct.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issue Tracker.
- Describe the bug and include steps to reproduce.

### Suggesting Enhancements
- Open an issue with the tag `enhancement`.
- Describe the feature and why it would be useful.

### Pull Requests
1. Fork the repo and create your branch from `master`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Make sure your code follows the existing style (run `npm run format`).

## Styleguide
- Use TypeScript for all new code.
- Follow the Prettier configuration.
- Write descriptive commit messages.

Thank you for your help!
